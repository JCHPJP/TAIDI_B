# 1.1.1 医保支付改革对中药行业的影响  

医保支付改革正成为推动中药行业发展的关键政策引擎，尤其在住院与门诊两大场景中展现出差异化激励路径。在住院DRG（疾病诊断相关分组）支付体系下，多地已推行针对中医类医疗机构的激励政策。以绍兴市为例，根据《关于进一步完善中医药医保支付政策的通知》，当地全面推行住院DRG 支付中医激励机制，依据中医类医疗机构的“中治率”设置激励系数，鼓励医院提高中医药治疗比例。这一机制有效提升了中药在住院治疗中的使用积极性，避免了因控费压力导致的中药应用萎缩。国家层面亦在2025 年10 月由国家医保局与国家中医药局联合发布《关于开展中医优势病种按病种付费试点工作的通知》，明确将在15 个左右省份或地级市启动试点，遴选如骨伤、肛肠、妇科等具有显著疗效的中医优势病种，实行按病种付费模式。试点期间支付标准不得低于改革前水平，充分保障中医机构利益，预计通过2～3 年实践积累经验后向全国推广，标志着中医药价值补偿机制进入系统化重构阶段。

# 1.1.2 道地药材保护政策实施进展  

道地药材保护政策近年来逐步构建起从资源普查到生态种植、产地加工的全链条制度框架，成为保障中药材质量与产业可持续发展的基石。2025 年3 月，国务院办公厅印发《关于提升中药质量促进中医药产业高质量发展的意见》，明确提出加强中药资源保护利用，编制专项发展规划，建立珍稀中药资源调查机制，并推进中药材生态种植养殖，全面推行《中药材生产质量管理规范》（GAP）。该文件作为国家级顶层设计，为各地政策实施提供了明确指引。同时，政策强调建设产地加工基地，强化流通与储备体系建设，推动中药材从“原字号”向“制成品”升级，提升附加值。

# 1.2 行业监管与标准化发展  

# 1.2.1 中药质量追溯体系建设  

中药质量追溯体系建设是保障中药材质量安全、提升行业透明度的关键举措。近年来，国家高度重视中医药行业的规范化发展，出台多项政策推动全产业链质量可追溯机制的建立。例如，《国务院办公厅关于提升中药质量促进中医药产业高质量发展的意见》（2025 年3 月15 日生效）明确提出要加强中药材流通和储备体系建设，指导建设产地加工基地，并推进中药资源数据库建设，为追溯体系提供制度支撑。同时，《关于质量基础设施助力产业链供应链质量联动提升的指导意见》也强调构建覆盖全链条、全流程的标准化体系，强化线 上联动，推动全链条质量协同升级。这些政策共同构成了中药追溯体系建设的顶层设计框架，旨在实现中药材‘来源可查、去向可追、责任可究’的监管目标。  

#  

当前，中药质量追溯体系已逐步覆盖种植、采收、加工、流通等关键环节。以 湖北省为例，省药监局武汉分局全面推进“清流”行动，通过下发《关于进一步加强药品信息化追溯体系建设工作的通知》，明确企业入驻标准和时间节点，推动药品流通环节追溯‘安全网’建设，确保数据采集的完整性与实时性。在技术层面，依托物联网、区块链与大数据平台，部分龙头企业已实现从田间到终端的全过程信息记录与共享。例如，亳州作为全国最大的中药材集散地，在国家政策支持下正加快现代化交易平台升级，推动标准化、数字化交易模式，提升供应链透明度。此外，《“十四五”中医药发展规划》亦提出支持建设现代化交易平台与追溯体系，引导行业向规模化、规范化发展。  

信息技术的应用显著提升了追溯系统的效率与透明度。通过统一编码规则与数据接口，企业可将药材种植环境、农残检测、炮制工艺、仓储物流等信息上链存证，形成不可篡改的数据链条。这不仅有助于监管部门实施精准溯源与风险预警，也增强了消费者对中药产品的信任度。更重要的是，追溯系统推动了全产业链质量责任的落实，促使各环节主体主动提升管理水平。然而，在实际推进过程中仍面临诸多挑战：一是标准不统一，不同地区、企业采用的技术平台和数据格式各异，导致信息难以互通；二是存在‘数据孤岛’现象，跨部门、跨系统间的数据共享机制尚未健全，影响整体协同效能。  

为此，亟需加强跨部门协作与平台互联互通。建议由国家中医药管理局牵头，联合农业农村部、市场监管总局等部门，建立统一的中药追溯标准体系和技术规范，推动国家级追溯平台与地方系统对接。同时，鼓励行业协会制定团体标准，引导企业接入公共追溯网络。通过政策激励与监管约束双轮驱动，逐步实现全行业数据开放共享，最终建成纵向贯通、横向协同的中药质量追溯生态体系。

# 1.2.2 经典名方审批政策优化  

经典名方审批制度的优化是我国中药传承创新战略的重要组成部分。过去，中药新药研发面临周期长、成本高、证据要求复杂等问题，制约了传统经方的现代化转化。为此，国家持续推进审批制度改革，简化审评流程，明确证据等级认定标准。例如，《国务院办公厅关于提升中药质量促进中医药产业高质量发展的意见》（2025 年3 月15 日生效）提出要以体制机制改革为保障，强化临床价值评估，构建中成药临床综合评价指标体系，为经典名方的研发与审评提供科学依据。同时，《关于促进中医药传承创新发展的实施意见》（浙江省义乌市，2024 年1 月1 日生效）也强调推进中医药传承创新发展，支持丹溪中医药文化品牌建设，反映出地方政府对经典名方价值挖掘的支持导向。  

政策对中药传承创新的支持机制日益完善，显著缩短了研发周期并降低了开发成本。相较于全新化学药，经典名方因其长期临床应用历史，具备“人用经验”优势，可在一定程度上减免非临床安全性试验和部分临床研究，从而加快上市进程。这一简化路径极大激发了企业的研发积极性，尤其对于拥有丰富古籍文献资源的中药老字号企业而言，成为其产品线拓展的重要突破口。例如，部分企业已基于《古代经典名方目录》开展制剂开发，利用现代提取技术和质量控制手段进行工艺优化，在保持疗效的同时提升稳定性和便携性，实现了传统智

慧与现代科技的融合。  

经典名方目录的动态更新机制进一步引导企业聚焦临床价值明确的产品开发。国家中医药管理局会同相关部门定期遴选具有广泛使用基础、疗效确切、安全性高的古代方剂纳入目录，为企业提供清晰的研发指引。这种“目录$^+$政策$^+$技术指导”的组合模式，不仅提高了研发的靶向性，也避免了低水平重复申报。企业可根据目录优先布局潜力品种，结合市场需求进行剂型改良或功能拓展，如将汤剂转化为颗粒剂、胶囊等更符合现代消费习惯的形式，增强产品的市场竞争力。  

审批制度改革与标准化体系建设之间呈现出显著的协同关系，正在重塑中药新药研发生态。一方面，审评标准的明确化倒逼企业在研发初期即建立严格的质量控制体系；另一方面，随着《中医药标准管理办法（征求意见稿）》的推进，国家标准、行业标准、团体标准和企业标准多层次体系逐步完善，为经典名方的质量一致性提供了技术支撑。两者结合，形成了“政策引导—标准护航—企业落地”的良性循环。未来，随着更多高质量证据的积累和真实世界研究方法的应用，经典名方的审评将更加科学化、国际化，有望推动一批具有全球影响力的中药新药问世。

# 3.1.1 AI 辅助诊断技术应用

# 3.2.1 创新中药制剂技术突破  

现代制剂技术的突破正显著推动中药产品的升级迭代，尤其在缓释、靶向输送与生物利用度提升方面取得实质性进展。以复方丹参滴丸为代表的现代中药产品，通过微丸包衣与控释技术实现了药物在体内的持续释放，延长药效时间并减少服药频次，提升了患者依从性。同时，靶向给药系统如脂质体、聚合物纳米粒等载体的应用，使活性成分能够更精准地作用于病灶部位，降低全身毒性，提高治疗指数。例如，部分企业已开展基于肿瘤微环境响应的中药纳米制剂研究，实现对癌细胞的选择性杀伤。  

新型辅料与纳米载体技术的进步有效改善了传统中药成分的溶解性与吸收效率。许多中药活性成分如黄酮类、皂苷类存在水溶性差、膜通透性低等问题，限制其口服生物利用度。通过引入环糊精包合、固体分散体、自微乳化系统等技术，可显著提升难溶性成分的溶出速率和肠道吸收。例如，人参皂苷Rg3 经纳米载体包裹后，其血药浓度-时间曲线下面积（AUC）较普通制剂提升超过2 倍。此外，聚乳酸-羟基乙酸共聚物（PLGA）等可降解材料作为纳米载体，不仅增强稳定性，还可实现缓释与靶向双重功能。

# 3.2.2 中药现代化提取工艺升级  

中药现代化提取工艺的升级是保障药材有效成分高效获取与产品质量稳定的核心环节。超临界流体萃取（SFE）、膜分离技术与连续逆流提取（CCLE）等先进工艺已在行业内逐步推广应用。超临界$\mathsf{C}\mathsf{O}_{2}$萃取利用高压下$\mathsf{C}\mathsf{O}_{2}$的高渗透性和溶解能力，可在低温条件下选择性提取脂溶性成分如挥发油、生物碱，避免  

热敏成分降解。膜分离技术则通过不同孔径的滤膜实现大分子杂质与小分子活性成分的高效分离，显著降低重金属与农残残留。连续逆流提取通过物料与溶剂的反向流动设计，大幅提升提取效率，缩短工艺周期，适用于大规模工业化生产。  

新工艺在提高目标成分提取率与减少杂质残留方面展现出显著优势。相较于传统煎煮法，超临界萃取对丹参酮IIA 的提取率可提升$40\%$以上，且无有机溶剂残留；膜分离技术应用于银杏叶提取物生产中，可将黄酮苷纯度提升至$95\%$以上，同时去除超过$80\%$的淀粉与蛋白质杂质。连续逆流提取相比静态浸提，溶剂用量减少$30\%–50\%$，提取时间缩短$60\%$，极大提升了资源利用效率。这些技术进步直接支撑了《中药材生产质量管理规范》的落地实施，强化了全产业链质量控制。  

工艺参数的精准控制对批次间一致性与产品质量稳定性至关重要。现代提取过程普遍集成在线监测系统（PAT），实时监控温度、压力、流速、pH 值等关键参数，确保每一批次产品的化学指纹图谱高度一致。例如，寿仙谷在其智能化制剂生产线中引入英克系统，实现全过程数据追溯与质量预警，其灵芝、铁皮枫斗颗粒获评“浙江省优质产品金奖”。数字化系统的应用不仅提升了合规水平，也为企业申报国家中药标准提供了可靠数据支持。  

先进提取技术的产业化需综合权衡设备投资、能耗控制与环保合规成本。超临界萃取设备初期投入较高，单套系统可达千万元级别，且运行需高压容器与精密控制系统，维护成本较大；膜分离虽节能但膜材料易污染，需定期更换，增加运营支出；连续逆流提取虽效率高，但对自动化控制系统要求严苛。尽管如此，在政策引导下，企业正积极布局绿色智能制造。《国务院办公厅关于提升中药质量促进中医药产业高质量发展的意见》明确推进中药工业数字化智能化发展，支持建设智能工厂。寿仙谷“年产2750 万袋粉剂、3100 万袋颗粒剂……智能化制剂生产线扩建项目”顺利通过验收，标志着高端提取工艺正加速向规模化、集约化转型。

# 4.1 产业链上下游图谱分析  

# 4.1.1 中药材种植与加工环节协同  

中药材种植与加工环节的协同是保障中药产品质量稳定性和提升产业附加值的关键路径。首先，种源选育与标准化种植基地建设为后续加工提供了均一、可控的原料基础。以樟树市为例，当地通过建设千亩以上种植示范基地3 个，实现中药材人工种植面积达20 余万亩，年总产值6.56 亿元，销售量达10,288 吨，显著提升了原料供给能力（《2023 年中医药行业研究报告“传统文化的传承与创新，过去与未来的相互融合”》）。在此基础上，推行《中药材生产质量管理规范》（GAP），强化生态种植与种子资源管理，确保药材从源头具备可追溯性与质量一致性。  

其次，产地初加工环节的规范化操作直接影响药材品质保持。干燥、净制、切制等工序若控制不当，易导致有效成分流失或微生物超标。政策层面，《国务院办公厅关于提升中药质量促进中医药产业高质量发展的意见》明确提出要指导建设产地加工基地，推动中药材现代种业发展，并研究制定中药材种子管理办法，为初加工标准化提供制度支撑。例如，宜春市推行“企业$^+$基地$^+$合作社$\cdot^{+}$农户”模式，实施“六统一”（统一种植标准、统一种子种苗、统一农资供应、  

#

# 4.1.2 中医诊所终端服务网络建设  

中医诊所作为中医药服务体系的终端服务节点，在基层医疗网络中发挥着不可替代的作用。根据《关于促进中医药传承创新发展的实施意见》，浙江省义乌市等地正推进标准化中医馆建设，提供中医预防保健、康复服务，并推广基层中医适宜技术，夯实基层服务能力。截至2025 年，河北广宗县塘疃中心卫生院已建成旗舰国医堂和村级中医阁，提供针灸、推拿、拔罐、刮痧、熏蒸等多种特色治疗项目，年门诊量突破5 万人次，真正实现“家门口”的中医药服务覆盖。  

智慧共享中药房的建设正推动处方流转、集中煎煮与配送服务的高效协同。山东省卫健委等五部门联合发布的《关于加快推进县级中医医院高质量发展的若干措施》明确提出，要提升中药药事保障能力，建设智慧共享中药房。此类平台依托信息化系统，实现区域内多家医疗机构处方实时上传、统一审方、集中煎配、物流配送，大幅提升服务效率与用药安全。例如，东城区通过“杏巷”建设联席会整合26 家单位资源，包括东单中医门诊部、同仁堂、长春堂等，构建专业服务网络，实现资源共享与服务协同。

# 5.2 品牌建设与国际化布局  

# 5.2.1 国际化标准认证进展  

近年来，中国中药企业加速推进国际化标准认证，逐步获得包括GMP（良好生产规范）、ISO 9001 质量管理体系以及有机产品认证在内的多项国际资质。以中之杰为例，该企业凭借其全产业链体系和39 项国家专利技术，已取得ISO9001 等国际认证，并以此为基础拓展海外市场，进入四省一市12 座城市的直营体系，成为区域性品牌走向全国化、国际化发展的典型案例。此类认证不仅提升了企业的内部管理标准，也显著增强了海外经销商与消费者对其产品质量的信任度。在欧盟市场，类似“土大妈”这样的农产品品牌通过获得出口认证，成功打通了高端农产品的国际市场通道，为中药类产品提供了可借鉴路径——即通过认证实现从产地到终端的透明供应链建设，强化品牌形象。  

然而，不同市场的认证标准存在差异，给中药企业带来挑战。例如，欧美地区对中药材的重金属残留、农药残留及微生物限量要求严于中国国内监管标准，且强调全过程可追溯性，这要求企业在质量管理体系建设上进行系统性升级。  

#  

此外，部分国家将中药归类为膳食补充剂或传统草药，适用不同的注册路径，导致企业在申报时需额外准备临床数据或传统使用证据，增加了合规成本。相较之下，国内GMP 主要聚焦于生产环节控制，而在环境可持续性、社会责任等方面尚未形成强制性要求，而这些正是国际有机认证或ESG 相关评估中的关键维度。  

尽管认证投入较高，涵盖检测费用、体系改造与第三方审计等，但其长期价值不可忽视。一方面，获得国际认证有助于企业突破贸易壁垒，如东盟、中东等地已将GMP 作为药品进口的基本门槛；另一方面，认证成为品牌溢价的重要支撑。宁波北仑通过推动“品字标浙江制造”认证，助力区域品牌走向高端化，体现了认证对品牌含金量的提升作用。对于中药企业而言，多国认证不仅是准入条件，更是塑造全球品牌形象的战略工具。下表总结了主要国际认证及其市场覆盖情况：  

| 认证类型 | 主要适用市 | 核心要求 | 对中药企业的意义 |
| -------- | ---------- | -------- | ---------------- |
| WHO-GMP  | 非洲、东南亚、中东 | 生产过程规范化、质量控制体系健全 | 满足发展中国家药品进口基本要求 |
| EU-GMP   | 欧盟成员国 | 全流程可追溯、数据完整性高 | 进入欧洲主流医药市场的通行证 |
| ISO 9001 | 全球通用 | 质量管理体系标准化 | 提升管理效率，增强合作方信任 |
| USDA Organic / EU Organic | 北美、欧洲 | 原料无化学农药、非转基因、生态种植 | 支持中药饮片及保健品高端化定位 |  

未来，随着国际社会对中国传统医学认可度提升，中药企业应以认证为抓手，推动质量管理体系与国际接轨，同时借助认证成果反哺国内品牌建设，实现内外市场协同发展。

# 5.2.2 海外市场准入策略优化  

中药企业进入海外市场面临复杂的法规准入门槛与注册审批流程，各国监管体系差异显著。在欧美市场，中药通常被归类为植物药、膳食补充剂或传统草药产品，需分别遵循美国FDA 植物药指南、欧盟传统草药注册程序（THMPD）等规定，涉及成分披露、安全性评估及标签合规等多项要求。而在东南亚、非洲等新兴市场，虽然部分国家承认中医药地位，但仍需完成本地药品注册、清关检验及分销许可等手续。政策层面，《商务部等9 部门关于拓展跨境电商出口推进海外仓建设的意见》明确提出支持企业提升合规经营水平，推动标准规则国际合作，为企业应对多元监管环境提供了政策引导。

# 6.1.1 成长期细分赛道识别

# 6.2 市场吸引力与风险评估  

# 6.2.1 政策支持下的发展机遇  

国家层面出台的高质量发展指导意见显著提振了中药行业的信心，并引导资源向优势企业集聚。2025 年3 月15 日发布的《国务院办公厅关于提升中药质量促进中医药产业高质量发展的意见》明确提出，以科技创新为支撑、体制机制改革为保障，加快构建现代化中药产业体系。该政策强调推进中药材生态种植养殖、建设高水平数字化车间和智能工厂，并支持中药大品种创新改良，为行业转型升级提供了顶层设计指引。在财政与金融支持方面，中央财政已统筹资金政策，鼓励银行业金融机构创新金融服务，重点支持设备更新和消费品以旧换新（国家发展改革委，2023 年10 月）。这一举措有效缓解了企业在技术改造中的资金压力。例如，昆药集团、太极集团等企业在国企改革背景下获得融资便利，推动产业链整合与产能升级。
