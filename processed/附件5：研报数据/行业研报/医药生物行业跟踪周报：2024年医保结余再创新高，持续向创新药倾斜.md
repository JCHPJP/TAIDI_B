# 2.   医保基金稳健运行，共济能力与使用质量显著提升  

# 2.1.   2024 年医保基金稳健运行，收支增速协调性改善  

# 2025 年3 月21 日，国家医保局发布《2024 年医疗保障事业发展统计快报》  

参保人数基本稳定，医保保持高覆盖率。医保参保人数由2015 年的66582 万人增加至2024 年的13269 万人，2024 年目前覆盖率$95\%$，医保参保规模整体保持稳定，覆盖面依然全面。考虑到2024 年8 月国务院办公厅发布关于健全基本医疗保险参保长效机制的指导意见，特困人员、低保户的资助投保有望落实，城乡居民参保人数有望进一步提升，医保覆盖率和带动医保基金收支规模预计将进一步提升。  

医保基金收支规模持续扩大，收支增速协调性增强。基本医疗保险基金（含生育保险）总收入由2014 年的12332 亿元增长至2024 年的34810 亿元，CAGR 为$9.89\%$；支出由2014 年的12170 亿元增长至2024 年的29676 亿元，CAGR 为$8.44\%$。近年来，医保基金保持收入增速略高于支出增速态势，主要得益于医保控费措施如集采、DRG/DIP、医保国谈等的良好执行。2023 年医保支出增速$15.46\%$，远高于收入增速$9.13\%$，一定程度上引发了关于医保基金穿仓的担忧，但2024 年收支增速分别为$3.9\%$和$5.2\%$，收支增速协调性增强。考虑到近年来多地为企业减负，允许缓缴企业负担的医保费用或降低费率的大背景，未来医保基金收支规模有望保持健康的增长态势。2014-2023 年医保基金均保持略有结存状态，截至2023 年底，医保基金累计结存47833.71 亿元，整体而言医保基金持续安全稳健运行。

# 2.2.   共济能力持续提升，医保资金使用效率优化  

统筹基金支出占医保基金整体支出比例提升，共济能力进一步加强。2024 年医保基金整体支出29675.92 亿元，其中统筹基金支出23824.42 亿元。统筹基金支出占医保基金整体支出的比例从2020 年的$77.54\%$提升至2024 年的$80.28\%$，2024 年占比相较2023年提升1.9 个百分点，表明医保基金的共济能力持续提升，医保资金的使用效率得到了优化。

# 2.3.   控费举措成效显著，腾笼换鸟效应提高医保基金使用质量  

DRG 后控费效应明显，次均住院费用显著下降。2021 年全面执行DRG 后，职工医保住院人次从2021 年的5900 万人次增加至2024 年的8652 万人次，但次均住院费用从2021 年的12948 元降至2024 年的11169 元，2024 年同比降幅$8.26\%$；居民医保住院人次从2021 年的15348 万人次增加至2024 年的20530 万人次，但次均住院费用从2021年的8023 元下降至2024 年的7295 元，2024 年同比降幅$4.94\%$。DRG 自2019 年起实行，随着改革推广深化，控费成效自2021 年起开始体现。截至2024 年底，按病种付费覆盖全国所有统筹地区，按病种付费的医保基金支出占统筹地区内住院医保基金支出超八成。116 个统筹地区上线按病种付费2.0 版分组方案，九成以上的统筹地区建立了特例单议、谈判协商机制，所有统筹地区建立了医保数据工作组。

# 3.3.   重要研发管线一览  

3 月20 日，迪哲医药（股票代码：688192.SH）宣布，公司将于当地时间3 月26 日-29 日在法国巴黎举行的2025 年欧洲肺癌大会（ELCC）上，以壁报形式首次展示其自主研发的两款源头创新药物舒沃哲$^\mathrm{\textregistered}$（通用名：舒沃替尼片）和高瑞哲$^\mathrm{\textregistered}$（通用名：戈利昔替尼胶囊）分别针对晚期非小细胞肺癌（NSCLC）靶向和免疫治疗耐药的最新研究进展。  

3 月19 日，全球临床试验收录网站clinicaltrials 显示，华东医药启动了小分子GLP-1 受体（GLP-1R）激动剂HDM1002 的首个III 期临床试验。这是第3 款进入III 期临床阶段的国产小分子GLP-1R 激动剂。 3 月18 日，药物临床试验登记与信息公示平台显示，石药集团启动了SYS6010的首个III 期临床试验。这是第二款进入III 期临床阶段的国产EGFR ADC。  

3 月17 日，据外媒Endpoints 报道，阿斯利康公布了其靶向B7-H4 的ADC 药物puxitatug samrotecan（P-Sam）在子宫内膜癌中的首次人体临床研究数据。结果显示，P-Sam 对这类患者具有良好的疗效和安全性，阿斯利康正在完善其III 期临床研究计划。  

3 月14 日，药物临床试验登记与信息公示平台显示，信达启动了一项在中国超重或肥胖合并代谢相关脂肪性肝病（MAFLD）受试者中比较玛仕度肽与司美格鲁肽有效性和安全性的多中心、随机、开放标签的III 期临床研究（GLORY-3）。  

3 月14 日，药物临床试验登记与信息公示平台显示，齐鲁制药启动了QL2108的III 期研究。QL2108 是首个进入III 期临床阶段的度普利尤单抗生物类似药。

# 6.   风险提示  

1、药品或耗材降价幅度继续超预期：  

国家加大集采范围，药品和耗材的集采带来了药品和耗材的大幅降价，随着国内医药和器材市场的竞争加剧，集采对降价的推动作用将会更加明显，未来药品和耗材集采 降价幅度可能会超预期，对企业的利润造成较大影响。  

2、医保政策进一步严厉等：  

医保局作为中国医药行业最大单一支付方，其支付政策变化可能直接导致相关公司盈利水平发生显著变化。  

3、产品销售及研发进度不及预期。
