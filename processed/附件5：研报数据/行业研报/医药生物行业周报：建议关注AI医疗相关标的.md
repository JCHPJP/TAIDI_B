# 投资建议  

在反腐和集采背景下，关注临床刚需的品种；除此之外，CXO 以及科研服务上游相关领域的需求正在逐步恢复，叠加去年基数相对较低，建议关注相关标的：三友医疗、爱康医疗、益诺思、阳光诺和、百普赛斯等。 建议关注与医药+AI 相关领域标的： 1、AI+分级诊疗，建议关注平安好医生等线上就诊标的；2、AI+体检：建议关注美年健康；3、AI+早筛：建议关注华大基因、康为世纪；4、AI+商业：建议关注连锁药店一心堂、健之佳等公司。

# AI 在医疗行业中的应用预计将更加深入  

截至2025 年2 月中旬，中国已有多家医院接入DeepSeek，完成本地化部署，智慧医疗进程加速。成都市第一人民医院、深圳大学附属华南医院、上海市第六人民医院、北京中医药大学深圳医院（龙岗）等多家医院已接入DeepSeek。DeepSeek 具备开源生态，引入MLA 技术，推理计算量更低，可以在医疗场景服务更多用户、医生。DeepSeek 将深度学习与强化学习融合，采用“分层次知识蒸馏”算法框架，既提升了模型在复杂场景中的泛化能力，又通过动态参数优化确保了垂直领域的精准适配。  

随着人工智能（AI）技术的不断发展和深入应用，特别是在Deepseek 等技术的推动下，AI 在医疗行业的潜力正逐步显现。尽管目前医疗AI 尚未完全实现大规模落地，但未来其在医疗领域的应用前景十分广阔。我们预计，AI 在医疗行业的落地应用将主要集中在以下四个方面：  

# 1. AI+ 分级诊疗  

分级诊疗一直是医疗行业的一个重要痛点，主要原因在于医疗资源的分布不均和错配问题。优质医疗资源往往集中在大城市和顶级医院，而基层医疗机构的资源相对匮乏，导致患者大量涌向大医院，进一步加剧了医疗资源的紧张和分配不均。AI 的引入有望改善这一局面。通过AI 技术的赋能，基层医疗机构可以获得智能化的辅助诊断工具，帮助医生快速识别患者病情的严重程度，并将患者分流到合适的医疗机构进行治疗。这不仅能够提高诊断效率，还能优化医疗资源的利用率，缓解大医院的压力。此外，AI 还可以通过远程医疗技术，将顶级专家的诊疗能力下沉到基层，为偏远地区的患者提供更高质量的医疗服务，从而实现真正意义上的分级诊疗。  

# 2. AI+ 体检  

体检是健康管理的重要环节，但传统体检模式往往存在数据解读不够深入、个性化服务不足等问题。AI 技术的加入可以改变这一现状。通过对体检数据的深度分析，AI 能够为患者提供更加精准、全面的健康评估和风险预测。例如，AI 可以根据患者的体检数据，结合其生活习惯、家族病史等信息，生成个性化的健康管理方案，帮助患者更好地预防疾病。此外，AI 还可以通过智能化的体检报告解读功能，让患者更直观地了解自己的健康状况，提升体检的服务体验。智能体检报告分析App 和健康管理平台正在逐步改变传统体检的服务模式。  

# 3. AI+ 早筛  

疾病的早期筛查是提高治愈率、降低医疗成本的重要手段。然而，传统的早筛技术往往面临成本高、效率低、解读复杂等问题，限制了其大规模推广。AI 技术的引入为早筛领域带来了变化。通过AI算法的优化，早筛检测的成本可以显著降低，同时检测的准确性和效率也得到了提升。例如，在癌症早筛领域，AI 可以通过对基因测序数据的深度学习和分析，快速识别潜在的疾病风险，并提供更精准的解读。这不仅能够帮助医生更早地发现疾病，还能让患者以更低的成本获得高质量的筛查服务。未来，随着AI 技术的进一步发展，早筛的覆盖范围将更加广泛，更多的疾病将能够通过AI 技术实现早期发现和干预。  

# 4. AI+商业  

通过AI 技术的赋能，医药商业行业可以实现智能化升级。例如，AI 可以通过分析消费者的健康数据和购买记录，提供个性化的药品推荐和健康管理建议，提升用户体验。同时，AI 还能够优化药品库存管理，预测市场需求，减少库存积压和浪费。此外，智能药品售卖终端和线上药品零售平台的结合，也将通过AI 技术实现更高效的运营模式，为消费者提供便捷的购药服务。
