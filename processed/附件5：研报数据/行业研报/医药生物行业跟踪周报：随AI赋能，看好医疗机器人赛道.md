# 2.   未来已来，机器人在医疗领域渗透率持续提高  

随着人口老龄化的加剧和慢性病、复杂疾病发病率的增加，医疗机器人市场需求将持续增长。特别是康复机器人和辅助机器人市场，将帮助更多患者恢复肢体功能、提高生活质量，并有效减轻医护人员的工作负担。同时人工智能、智能传感等技术的快速发展与深度应用，医疗机器人产业将迎来更多的技术创新与升级。我们主要将医疗机器人分为手术机器人、康复机器人及医疗物流机器人。  

# 2.1.   手术机器人  

手术机器人，作为医疗机器人的重要子领域，融合了临床医学、生物力学、机械学、计算机科学以及微电子学等多个学科和高新技术，主要应用于手术影像导航定位及微创手术领域。目前，手术机器人的研究和应用已经非常的广泛。在临床上，用机器人开展手术基本以及覆盖了全部外科领域，像在普通外科、心脏外科、泌尿外科、妇科、耳鼻喉科、整形外科、神经外科和骨科等都有很好的应用案例。  

# 手术机器人分类及治疗范围

## 腔镜手术机器人
- 外科、泌尿外科、妇科、儿科等
- 治疗症状：腹腔、盆腔、胸腔等器官切除等

## 骨科手术机器人
- 骨科
- 治疗症状：髋关节、膝关节、脊柱等置换手术、脊柱手术等

## 泛血管手术机器人
- 心血管科、脑科神经外科等
- 治疗症状：心脏、心血管、脑部等介入手术、血管介入手术等

## 经自然腔道手术机器人
- 外科、呼吸内科、消化内科等
- 治疗症状：支气管检查、胃镜检查等

## 经皮穿刺手术机器人
- 肿瘤科、泌尿外科等
- 治疗症状：检测早期肺癌、清除肾结石等

## 神经外科手术机器人
- 检验科、神经内科、神经外科等
- 治疗症状：癫痫治疗、帕金森治疗等

## 其他手术机器人
- 各类科室
- 除了基础六种分类外的创新手术机器人类型：分离手术机器人、消融机器人等

根据 ReportLinker 发布的“2023 年手术机器人全球市场报告”数据，全球手术机器人市场预计将从2022 年的56.8 亿美元增长到2023 年的66.8 亿美元，复合年增长率（CAGR）为$17.8\%$，预计到2027 年，手术机器人市场将达到127.3 亿美元。  

技术进步是手术机器人市场中越来越流行的一个关键趋势。手术机器人行业的主要公司正在专注于开发新技术，以提高患者的疗效并巩固他们在市场上的地位。例如，2021年11 月，英国的医疗设备制造公司Smith+Nephew 推出了CORI 手持机器人，这是一种用于全膝和部分膝关节置换手术的一站式机器人辅助手术系统。它配备了一个3D 术中成像系统和机器人雕刻工具，使医生能够测量、计划和进行膝关节手术。该系统是一个紧凑且完全移动的解决方案，设置时间最少，并可以从一个手术室移动到另一个手术室，以优化手术单位的患者流动。  

# 2.2.   康复机器人  

根据国家统计局数据显示，从2016 年到2022 年，我国65 岁及以上人口已从1.5亿增长至2.1 亿，老年人群在康复治疗服务中占比接近$30\%$。  

康复机器人的核心技术包括传感系统、控制系统和驱动系统，这三大系统构成了康  

复机器人的基本功能框架。而仿生设计则根据人体工程学，设计更符合人体自然运动习惯的部件，提高了患者的舒适度和安全性。此外，续航能力的提升也是康复机器人未来发展的关键，尤其是在探索新型能源利用方面，如何将生物能等新技术融合进系统设计中，成为了重要的研究方向。  

政策环境方面，国家近年来发布了一系列政策，意在推动康复机器人的智能化发展，例如加强对人机交互、脑机接口及柔性技术的支持，鼓励科研机构与企业合作开发新型康复设备。同时，针对康复医疗服务机构的资源投放也在逐步增加，政策上鼓励民间资本的参与和医保覆盖范围的扩展，为行业带来了更多的资金与活力。  

# 2.3.   医疗物流机器人  

医疗物流机器人：医疗物流机器人正被应用于广泛的领域，作为未来定位医院和其他医疗设施的关键手段。医院机器人应用于各个领域，通过提高工作效率、降低成本负担和工人受伤的可能性，在病房之间运输药品和标本、访客引导和定期检查患者健康状况的访客，以及运输洗衣、邮件、食品等方面起到了重要作用。  

# 2.4.   投资建议  

手术机器人：微创机器人、天智航、三友医疗康复机器人：诚益通、翔宇医疗、伟思医疗 医疗物流机器人：健麾信息、艾隆科技

# 6.   风险提示  

1、药品或耗材降价幅度继续超预期：  

国家加大集采范围，药品和耗材的集采带来了药品和耗材的大幅降价，随着国内医药和器材市场的竞争加剧，集采对降价的推动作用将会更加明显，未来药品和耗材集采 降价幅度可能会超预期，对企业的利润造成较大影响。  

2、医保政策进一步严厉等：  

医保局作为中国医药行业最大单一支付方，其支付政策变化可能直接导致相关公司盈利水平发生显著变化。  

3、产品销售及研发进度不及预期。
