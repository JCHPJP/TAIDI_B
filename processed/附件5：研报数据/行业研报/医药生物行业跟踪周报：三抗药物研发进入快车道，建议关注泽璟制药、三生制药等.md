# 2.     三抗研发进入快车道，重点关注三生制药，泽 璟 制药  

三特异性抗体（Trispecific Antibodies, TsAbs）是一类能够同时结合三个不同抗原靶点的工程化抗体。与传统的单特异性或双特异性抗体相比，三抗具备更复杂的靶向能力，能够协同作用于肿瘤细胞表面抗原（TAA）、免疫细胞激活受体及共刺激分子，从而优化免疫细胞的杀伤功能，并减少耐药性的发生。此外，三抗还可以通过桥接不同的信号通路来增强免疫应答，使其在癌症免疫治疗中展现出更高的临床应用潜力。目前，全球尚无三特异性抗体药物获批上市，不过已有多家企业布局，并研发设计出具有迭代优势的靶点组合药物。Harpoon（24 年被MSD 收购）、赛诺菲、强生是这一赛道的先行者，国内未上市药企中恩沐生物、嘉和生物、普米斯以及上市药企先声药业、泽璟制药、基石药业、信达生物、三生制药等亦有相关药物布局。

# 2.1.1.   IgG-like 形式  

IgG-like 形式的三抗基于天然 IgG 抗体的结构框架，包含 Fc 区域，能够通过 Fc 受体（FcR）介导效应功能（如抗体依赖性细胞介导的细胞毒性，ADCC）并延长半衰期。此类抗体通常采用 $_{2+1}$ 结构或 Fc 端接设计：  

$_{2+1}$ 结构 ：两个 Fab 臂分别靶向两个抗原，第三个结合位点通过额外的结构域（如单链可变片段  scFv  或  Fab  片段）融合至抗体的  N  端或  C  端。例如 泽璟制药的ZG006，靶向 DLL3、CD3 和 DLL3 通过 $_{2+1}$ 结构增强 T 细胞与肿瘤细胞的结合，并提供共刺激信号；  

Fc 端接结构 ：额外的结合域（如 scFv）直接融合至 Fc 区，实现三重靶向能力。例如强生的JNJ-8543，二硫键链接两个臂分别为抗CD3 臂和抗CD79b 臂，下方Fc 链连接一个抗CD20 臂，以促进 T 细胞活化。

IgG-like 形式的三特异性抗体的优势在于其基于天然 IgG 框架设计，具有较高的结构稳定性与生产可行性， Fc 区不仅能通过与Fc 受体（FcRn）结合减少抗体降解，延长半衰，还可介导抗体依赖性细胞介导的细胞毒性（ADCC）及补体依赖性细胞毒性（CDC），增强抗肿瘤效应。局限性主要来自此类抗体的额外结合域，可能因空间位阻干扰 Fc 与 FcR 的正常相互作用，同时若结合域来源于非人源序列，可能增加患者免疫原性风险。  

# 2.1.2.   non-IgG-like 形式  

non-IgG-like 形式由抗体来源的构建模块通过柔性linker 连接组成，最简单的排列是“串珠”形式。这些结构域从分子量较高到较小可以是抗原结合片段（Fab）、单链可变片段（scFv）或单域抗体（sdAb）。通过构建模块的多聚化，可灵活调节抗体的大小、形状及价态，从而优化其半衰期与肿瘤渗透能力。  

无Fc 区的non-IgG-like 形式三抗药物通过将三个不同特异性的构建模块组装成单一多肽链，不仅规避了Fc 区域可能引发的非特异性免疫激活，还显著降低了分子量，使其更易穿透肿瘤组织并提升局部药物浓度。此外，这类抗体结构简单，可采用大肠杆菌等原核系统高效表达，显著降低了生产成本与工艺复杂性。然而，其无Fc 区的设计虽避免了免疫原性风险，却导致半衰期大幅缩短，需频繁给药以维持有效血药浓度，同时因缺乏Fc 受体介导的ADCC 及CDC 效应，其直接激活免疫系统的功能受到限制，需依赖其他机制（如共刺激信号增强）弥补这一缺陷。同时由于缺乏Fc 区域，non-IgG-like 抗体无法通过传统的Protein A（ProA）亲和层析纯化，需依赖其他亲和标签（如His-tag 或特定抗体结合蛋白），增加纯化步骤的复杂性和成本。

目前尚未有上市的三抗药物，进度最快的是默沙东（MSD）斥资约30 亿美元收购EyeBiotech 获得的治疗糖尿病黄斑水肿（DME）的Restoret（Fzd4;LRP5;LRP5），处于临床2b/3 期。处于临床2 期的药物有辉瑞的用于治疗特应性皮炎的PF-07264660（IL-33;IL-4;IL-13）和PF-07275315（TSLP;IL-4;IL-13）以及赛诺菲从Innate Pharma 购得的用于治疗R/R AML、B-ALL 以及高危MDS 的bexatamig（NKp46;CD123;CD16）。

5 月11 日，礼来公布了SURMOUNT-5 研究的详细结果。该研究是一项开放标签 IIIb  期临床试验，旨在针对肥胖或伴有至少一种合并症（如高血压、血脂异 常、阻塞性睡眠呼吸暂停、心血管疾病）的超重成人（且不合并糖尿病）患者，评估GLP/GLP-1 双受体激动剂替尔泊肽（tirzepatide）与GLP-1 单受体激动剂司美格鲁肽（semaglutide）的有效性与安全性。

# 5.   风险提示  

# 1、药品或耗材降价幅度继续超预期：  

国家加大集采范围，药品和耗材的集采带来了药品和耗材的大幅降价，随着国内医药和器材市场的竞争加剧，集采对降价的推动作用将会更加明显，未来药品和耗材集采 降价幅度可能会超预期，对企业的利润造成较大影响。  

2、医保政策进一步严厉等：  

医保局作为中国医药行业最大单一支付方，其支付政策变化可能直接导致相关公司盈利水平发生显著变化。
