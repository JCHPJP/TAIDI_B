# 1.1.基因疗法产品已报产，掘金广阔慢病市场  

严重下肢缺血（Critical limb ischemia, CLI）是下肢外周动脉疾病（Peripheral arterydisease，PAD）进展至最为严重的缺血阶段，下肢动脉堵塞限制血液流动，临床表现为行走能力下降、静息痛（持续2 周以上）、溃疡和坏疽，严重影响患者的生活质量，部分患者甚至只能面临截肢或死亡。  

根据国家心血管病中心发布的《中国心血管健康与疾病报告2023》，推算目前中国PAD 患者约4530 万人，此外根据《基因治疗性血管生成在下肢缺血性疾病中的研究进展》（桂亮等），在PAD 患者中约有$10\%$患者最终可发展为CLI，我们预计国内CLI人群在数百万量级，且随着人口老龄化以及高血压、高血脂和糖尿病等风险因素的增多，预计下肢缺血患病率正在逐渐上升。  

CLI 目前尚无彻底治愈的药物，通过血管腔内介入或外科手术进行血运重建是目前治疗CLI 的最佳治疗方案。然而《基因治疗性血管生成在下肢缺血性疾病中的研究进展》（桂亮等）显示，仍有$30\%$患者由于高龄、合并疾病等原因不能耐受手术，其他治疗手段如药物治疗（抗凝药物、血管扩张药物）仅为对症治疗，此外截肢为最终治疗方式。  

# 严重下肢缺血的治疗流程

## 一、诊断与评估
- 静息痛
- 局部缺血病变、坏疽

## 二、疼痛控制
- 吗啡
- 局部伤口处理、感染治疗（抗生素）

## 三、紧急血运重建
- 可行
  - 腔内血运重建
    - 血流动力学评估
      - 良好
        - 控制心血管危险因素、消瘦、鞋子改造（去除病变部位压力）、监测
      - 不佳
        - 控制心血管危险因素、疼痛控制（吗啡）、局部伤口处理
        - 抗血小板/前列腺素类药物
        - 截肢、康复治疗
    - 失败、不适合腔内血运重建
      - 外科手术血运重建
        - 重新治疗（腔内或开放手术）

- 不可行
  - 重新治疗（腔内或开放手术）

# 1.2.前瞻布局镇痛创新靶点，护城河不断拓宽  

在镇痛新药开发领域，靶点主要包括基于阿片受体的疼痛与镇痛靶点（主流方向），阿片类镇痛药由阿片受体家族4 个亚型介导，即$\upmu$、δ、κ和NOP 受体。其中，$\upmu$阿片受体被公认是镇痛和副作用主要靶点，该受体与药物结合后能激活多条下游信号通路，在产生强效镇痛作用的同时伴随出现过度镇静、呼吸抑制、便秘、耐药、成瘾等不良反应。现阶段，新型阿片类镇痛药物的研发主要集中在三条路线：偏向性激动策略、外周阿片受体激动策略、“一药多靶”策略。  

此外如大麻素受体、 $\propto\!\mathscr{Q}$  肾上腺素能受体、趋化因子受体、血管紧张素2 型受体以 及一些离子通道被发现具有镇痛作用，为开发新型镇痛药物提供可能性。

# 1.5.P2X3 拮抗剂临床推进，布局难治性慢性咳嗽

# 3.风险提示  

1）行业竞争加剧风险：随着我国医药产业的不断创新与进步，不排除持续有新进竞争者加入从而造成竞争加剧的风险；  

2）政策变化风险：分级诊疗、医保支付改革持续成为医改重心，医保控费大趋势、带量采购等让医药产业中下游的收入利润有承压风险；
