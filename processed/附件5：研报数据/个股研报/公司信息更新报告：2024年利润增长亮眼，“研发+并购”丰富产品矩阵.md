# $\bullet$ 利润增长亮眼，维持“买入”评级  

公司2024 年实现营收35.00 亿元（同比$+5.72\%$，下文皆为同比口径）；归母净利润7.23 亿元（$\langle+27.19\%\rangle$）；扣非归母净利润6.42 亿元（$\cdot+20.20\%$）。2025Q1 实现营收10.21 亿元（$\cdot+12.29\%$）；归母净利润2.17 亿元（$\cdot+13.89\%)$）；扣非归母净利润2.04 亿元（$\cdot+12.38\%.$）。盈利能力方面，2024 年毛利率为$75.82\%$（$'+2.53\mathsf{p c t})$），净利率为$20.64\%(+3.48{\mathrm{pct}})$）。费用方面，2024 年销售费用率为$45.62\%(+0.20\mathrm{{pct})}$）；管理费用率为$4.76\%$（-0.50pct）；研发费用率为$3.76\%$（$+0.45\mathsf{p c t})$）；财务费用率为$-0.54\%$（$(+0.52\mathrm{{pct})}$）。我们看好公司贴剂核心业务的发展以及并购银谷制药带来的成长空间，同时考虑到潜在的费用支出，上调2025 年、下调2026 年并新增2027 年盈利预测，预计2025-2027 年归母净利润分别为$8.31/9.55/10.69$ 亿元（原预计为$8.28/9.69$ 亿元），EPS 为$1.47/1.68/1.89$ 元，当前股价对应PE 为$15.6/13.6/12.1$ 倍，维持“买入”评级。  

# $\bullet$   贴剂、片剂及胶囊剂营收稳健提升，软膏剂小幅承压  

分产品来看，2024 年公司贴剂产品实现营收21.15 亿元（$(+3.35\%)$）、片剂营收3.55亿元（$\langle+35.60\%]$）、胶囊剂营收7.52 亿元（$\zeta{+5.66\%}$）、软膏剂营收1.28 亿元（$(-7.09\%)$）、其他产品营收1.48 亿元（$\langle.1.20\%.$）。毛利率方面，贴剂产品毛利率为$78.95\%

$（$'+3.27{\mathrm{pct}}\,)$）、片剂毛利率为$69.94\%$（$+4.32\mathsf{p c t}\,)$、胶囊剂毛利率为$75.79\%$$(+1.13\mathrm{{pct}})$）、软膏剂毛利率为$75.17\%$（$\mathsf{\dot{+0.22p c t}})$）。  

$\bullet$ 研发、并购双引擎驱动，产品线持续补充丰富  

研发创新方面，2024 年公司持续围绕骨科、呼吸、皮肤、消化系统等领域，聚焦大病种、大品种开展布局，取得孟鲁司特钠咀嚼片、盐酸二甲双胍缓释片、莫 匹罗星软膏药品注册证书。并购方面，公司2024 年12 月与银谷控股集团等签订《收购意向书》，截至2025Q1 已完成银谷制药的股权交割，此次收购为公司补充了具有较高技术门槛的鼻喷剂、吸入剂等外用制剂剂型，进一步丰富产品矩阵。  

 风险提示：市场竞争加剧，产品销售不及预期，新产品研发进程不及预期。
