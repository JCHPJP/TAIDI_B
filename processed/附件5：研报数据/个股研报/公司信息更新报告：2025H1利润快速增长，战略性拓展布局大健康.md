# $\bullet$ 利润稳步快速增长，维持“买入”评级  

公司2025H1 实现营收15.99 亿元（同比$+11.99\%$，下文皆为同比口径）；归母净利润3.74 亿元（$\cdot+26.16\%.$）；扣非归母净利润3.72 亿元（$\langle+27.24\%$）。2025Q2单季度实现营收7.75 亿元（$\mathsf{\zeta}_{+2.60\%.}$）；归母净利润1.92 亿元（$\cdot+25.20\%.$）；扣非归母净利润1.90 亿元（$\cdot+28.30\%$）。从盈利能力来看，$2025\mathrm{H}1$ 毛利率为$63.01\%$（$(+0.39\mathsf{p c t})$），净利率为$23.71\%$（$_{+2.71}$pct）。从费用端来看，2025H1 销售费用率为$31.23\%$（-2.95pct）；管理费用率为$3.03\%$（-0.71pct）；研发费用率为$1.58\%$（-1.16pct）；财务费用率为$0.22\%$（$+0.62$pct）。我们看好公司乌灵及百令系列产品未来的成长空间，维持2025-2027 年盈利预测，预计2025-2027 年归母净利润分别为$6.67/8.45/10.67$ 亿元，EPS 为$0.95/1.20/1.52$ 元/股，当前股价对应PE 为$21.3/16.8/13.3$ 倍，维持“买入”评级。  

# $\bullet$ 乌灵及百令系列产品营收稳健增长，多数产品毛利率稳中有进  

2025H1 公司乌灵系列营收8.97 亿元（$(+7.23\%)$），毛利率$86.66\%$（-0.11pct）；百令系列营收1.13 亿元（$\cdot+38.51\%$），毛利率$63.95\%$（$\cdot+2.58$pct）；中药饮片营收3.49 亿元（$(.10.10\%)$），毛利率$28.25\%$（$\cdot6.91$1pct）；中药配方颗粒营收0.74 亿元（$(+56.60\%$），毛利率$62.56\%$（$\mathsf{:+2.99p c t.}$）；其他营收1.62 亿元（$(+134.48\%)$）。
