# $\bullet$   营收利润承压，看好阶段承压后的长期发展潜力，维持“买入”评级  

公司2025H1 实现营收56.58 亿元（同比$-27.63\%$，下文皆为同比口径）；归母净利润1.39 亿元（$(.71.94\%)$）；扣非归母净利润1.20 亿元（$(\AA.74.52\%$）。从盈利能力来看，2025H1 毛利率为$27.70\%$（-19.01pct），净利率为$2.52\%$（-3.91pct）。从费用端来看，$2025\mathrm{H}1$ 销售费用率为$13.99\%$（-17.66pct）；管理费用率为$6.10\%$$(+1.82\mathsf{p c t})$；研发费用率为$2.43\%\left(+0.94{\tt p c t}\right)$；财务费用率为$1.21\%\left(+0.45\mathsf{p c t}\right)$）。考虑到公司$2025\mathrm{H}1$ 营收利润承压主要受相关政策以及部分产品社会库存持续消化等因素影响，我们下调2025-2027 年盈利预测，预计2025-2027 年归母净利润分别为$4.59/5.48/6.54$ 亿元（原预计为$5.53/6.22/7.00$ 亿元），EPS 为$0.82/0.98/1.17$元/股，当前股价对应PE 为$28.2/23.6/19.8$ 倍，我们看好公司阶段承压过后的长期发展潜力，维持“买入”评级。  

# $\bullet$ 中药材资源营收稳健，医药工业和商业板块阶段承压  

分行业来看，2025H1 公司医药工业营收28.42 亿元（$\langle.44.03\%.$），医药商业营收33.72 亿元（$\cdot9.87\%$），中药材资源营收4.93 亿元（$\dot{}{,}{+0.55\%}.$），大健康及国际营收0.95 亿元（$\langle.64.15\%$），服务业及其他营收0.01 亿元（$\dot{}.63.95\%$）。

2025H1 公司持续优化营销组织架构，加快销售模式转型，优化营销人员薪酬结构和考核口径，激发营销队伍活力。同时，公司以市场为导向，以临床价值为核心驱动学术营销转型，学术推广体系战略升级，完成顶层设计，数字化医学平台正式立项。此外，终端动销方面也取得阶段性成果突破，（1）等级医院着力市场准入，新增三级医院超400 家，二级医院超1100 家；（2）连锁药店聚焦百强，强动销降库存；（2）基层医疗着力市场开拓，建网络拓渠道，新开发诊所3.9万家，打造样板诊所1729 家；（4）线上销售升级平台合作，新零售重突破；（5）终端团购海外做好补充，升级优化国际产品矩阵。
