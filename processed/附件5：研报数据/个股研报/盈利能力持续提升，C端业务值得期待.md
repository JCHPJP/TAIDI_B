# 佐力药业(300181)  

# 盈利能力持续提升，C 端业务值得期待  

# l业绩符合预期，盈利能力持续提升  

公司发布2025 年三季报，业绩符合预期。2025Q1-3 公司实现营业收入22.8 亿元（$(+11.48\%)$）；归母净利润5.1 亿元（$(+21.00\%)$），扣除股权激励和员工持股计划2291.34 万元影响，公司归母净利润为5.33 亿元（$(+26.43\%)$）；扣非净利润5.09 亿元（$\cdot+22.\,51\%)$），经营现金流净额2.66 亿元（$'+24.~23\%.$）。  

分季度来看，2025Q1 、Q2、Q3 营业收入分别为8.24 亿元（$(+22.\,52\%)$）、7.75 亿元（$(+2.60\%)$）、6.8 亿元（$(+10.32\%)$），归母净利润分别为1.81 亿元（$\langle+27.\,20\%\rangle$）、1.92 亿元（$\cdot+25.\,20\%\,\$）、1.36 亿元

（$(\,+8.\,81\%\,)$），扣非净利润分别为1.81 亿元（$(+26.~16\%)$）、1.9 亿元

（$(+28.\,30\%)$）、1.37 亿元（$\langle+11.\,28\%.\rangle$）。  

从盈利能力来看，公司盈利能力持续提升，2025Q1-3 毛利率为

62.$03\%$（$(+0.\,42\mathsf{p o t})$），归母净利率为$22.\,37\%\ (+1.\,76{\tt p o c t})$），扣非净利率为 $22.\,31\%\ \left(+2.\,01\,\mathsf{p c t}\right)$ 。  

费率方面，公司提质增效成果显著，费用率呈下降趋势。2025Q1-3销售费用率为$30.\,99\%\,(-2.\,46{\tt p o c t})$，管理费用率为$3.\,41\%\,(-0.\,37\mathsf{p c t})$，研发费用率为$1.\,83\%\,\,(-0.\,65\mathsf{p o c t})$，财务费用率为$0.\,29\%\ (+0.\,63\mathsf{p c t}\,)$。  

# $\bullet$百令系列、中药配方颗粒业务表现突出  

拆分来看，2025 年前三季度乌灵系列营业收入同比增长$6.\,98\%$；百令系列营业收入同比增长$29.58\%$；中药配方颗粒营业收入同比增长$57.41\%$；中药饮片营业收入同比减少$13.\,68\%$；佐力医药公司商业销售收入同比增长$113.68\%$。  

# $\bullet$深化C 端布局，院外市场有望走出新成长曲线  

公司深化“一路向$\complement^{\prime\prime}$，打造院外市场新标杆，一方面，外拓市场，乌灵胶囊推出108 粒包装，与头部连锁形成战略合作，打造慢病管理新模式；持续拓展B2C、O2O 及互联网主流电商平台，建设新媒体矩阵；另一方面，内强根基，OTC 事业部合并销售与市场职能，实现组织升级，资源协同，开展管理层及一线人员专项培训，打造高执行力团。院外市场有望走出新成长曲线。
